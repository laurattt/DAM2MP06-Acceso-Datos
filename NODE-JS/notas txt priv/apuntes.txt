PROMESAS EN JAVASCRIPT
Una promesa es un objeto que representa un valor en el futuro / un compromiso de entregar un resultado 

    - Es async (trabaja en segundo plano y no bloquea ejecución)

    - ESTADOS:
        - RESOLVE -> resuelta(devuelve el valor dado)
            var original = Promise.resolve(33);
            var cast = Promise.resolve(original);
            cast.then(function (value) {
            console.log("valor: " + value);
            });
            console.log("original === cast ? " + (original === cast));

            // registros, en orden:
            // original === cast ? true
            // valor: 33

        - REJECT -> error, ha sido rechazada 

        - PENDING -> pendiente de realizar (?)

    - fetch(): Método que inicia una petición HTTP (por defecto GET) y devuelve una promesa que se resuelve en un objeto
      request HTTPs -> metodo(POST) + URI
